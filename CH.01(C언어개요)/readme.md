# C언어 개요
---
C언어란
---
> C언어 실행과정<br>
```
01. 한글 영어 등등 Source code를 작성한 후 실행(run) 명령
02. 컴파일(Complie) 과정 실행 == 해석기
03. 디버그(Debug)과정 실행 == 오류 분석기
                            -- 버그 : 시스템상의 오류
                            -- 디버그 : 오류를 잡는 행위 -> 개발자에게 오류나 비정상적인 연산 등을 발견하고 알려주는 행위
04. Run(실행과정)
```
> C언어의 기본 구조<br>
```
01. 함수(Function) == '기능'
02. 함수 종류
    1) 메인(Main)함수 : 최초로 실행되는 함수, 단 하나만 존재해야됨 -> 같은 프로젝트안이라면 다른 소스파일일지라도 Main함수는 하나만 있어야 됨을 의미
    2) 라이브러리 함수 : 개발편의를 위해서 미리 만들어져 제공되는 함수
                        ex) printf() : 모니터 출력함수, scanf() : 키보드 입력함수
    3) 사용자 정의 함수 : 개발자가 정의한(만든) 함수
03. 작성 기본 구조
    1) #include <stdio.h>      -> #inculde == 전처리문
                               -> 전리리문 이란 : 컴파일 과정 이전에 미리 처리되는 문장
                               -> <studio.h> == 헤더파일
                               -> (Standard input output, 즉 표준 입출력 파일)
                               -> 헤더 파일 : 컴파일러에 의해 자동 포함된 소스코드 파일
                               -> 쉽게 말해 헤더 파일은 기본적인 함수를 가지고 있는 파일 즉, 헤더 파일로 정의된 함수를 사용하려면 해당 헤더파일이 입력되어있어야 사용이 가능
    2) int main(void) {        -> 함수의 헤더(int        main        ()  )        { : 함수의 바디(실질적 실행하는 내용)
                                            반환자료형   함수명      매개변수
    3) printf()                -> 모니터 출력 라이브러리 함수
                               -> ;(세미콜론) : 코드 라인의 마침표 역할
    4) return 0;               -> 반환값 0(함수가 종료되었을 때 전달하는 값)
                               -> 의미 : 함수가 무사히 종료되었다
    5) }                       -> 함수의 끝, 함수의 몸체 끝
```
